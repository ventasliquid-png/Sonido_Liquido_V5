# MANUAL DE USO PRELIMINAR - SONIDO LÍQUIDO V5

## 1. ALTA DE CLIENTES
El proceso de alta se divide en pestañas para organizar la información:
- **Pestaña 1 (Legal):** Ingrese los datos fiscales (CUIT, Razón Social). El sistema valida el CUIT automáticamente.
- **Pestaña 2 (Sucursales):** Aquí se definen los domicilios de entrega. Es obligatorio tener al menos uno.

## 2. LOGÍSTICA
- **Selector de Transporte:** Cada domicilio debe tener un transporte asignado.
- **Retiro en Local:** Si el cliente retira la mercadería, seleccione la opción "RETIRO EN LOCAL" en el campo de transporte. Esto es fundamental para la hoja de ruta.

## 3. ATAJOS DE TECLADO (Productividad)
- **F4 (Creación Rápida):** Si no encuentra un valor en un selector (ej: Localidad, Ramo), presione F4 para crearlo sin salir de la carga del cliente.
- **F10 (Guardar):** Guarda los cambios y cierra la ventana actual. Úselo siempre para finalizar.

## 4. SMART CUIT
- **Duplicados:** El sistema PERMITE CUITs duplicados para casos especiales (ej: UBA, Municipalidades).
- **Validación:** Si ingresa un duplicado, verá una alerta visual "A Revisar". Esto no bloquea la carga, pero marca el registro para auditoría posterior.

## 5. AGENDA Y CONTACTOS
- **Nuevo Contacto:** Desde la pestaña "Agenda", use el botón "+ Nuevo Contacto" o presione F4.
- **Roles:** Puede asignar roles (Ventas, Cobranzas, etc.). Si el rol no existe, escríbalo y presione "➕ Nuevo (F4)".
- **Edición:** Use el icono del lápiz para modificar datos de un contacto existente.
- **Copiar Datos:** Pase el mouse sobre el email o teléfono de un contacto para ver una ficha con el dato completo y un botón para copiarlo al portapapeles.
